Requirement 1: Create Campaign with Parameters

User Story: As a user, I should be able to create campaign in the system, and it should be created in 3rd party system too, with given parameters: Campaign name, Start & End Date, Budget, Campaign goal, Campaign time zone, Campaign frequency cap

What Can Be Done:
  • Campaign Name: ‚úÖ Fully supported
  • Start & End Date: ‚úÖ Fully supported
  • Budget: ‚ö†Ô∏è Depends on Campaign Budget Optimization (CBO) - With CBO enabled, budget is set at campaign level. Without CBO, budget must be set at ad set level. Workaround: Enable CBO by default for campaign-level budget control, or manage budgets at ad set level when CBO is disabled.
  • Campaign Goal: ‚úÖ Fully supported (traffic, conversions, etc.)
  • Time Zone: ‚ùå CANNOT be set per campaign - Facebook sets timezone at account level only. Workaround: Display timezone as read-only information from account settings.
  • Frequency Cap: ‚ùå CANNOT be set at campaign level - Facebook requires frequency caps at ad set level. Workaround: Automatically create default Ad Set with frequency cap settings when creating campaign.
‚ö†Ô∏è Partially Implementable

Requirement 2: Edit Campaign Settings

User Story: As a user, I should be able to edit campaign settings

What Can Be Done:
  • Campaign Name: ‚úÖ Can be updated anytime
  • Budget Amount: ‚úÖ Can be updated (with some restrictions for active campaigns)
  • Start/End Dates: ‚úÖ Can be updated (with constraints for running campaigns)
  • Campaign Status: ‚úÖ Can be changed (ACTIVE/PAUSED/ARCHIVED)
  • Frequency Cap: ‚úÖ Can be updated (through ad set level changes)
  • Campaign Objective: ‚ùå CANNOT be changed after creation - Facebook makes this immutable. Workaround: Create new campaign with desired objective, migrate settings, and archive old campaign.
  • Budget Type: ‚ùå CANNOT switch between daily/lifetime easily - Requires campaign pause and recreation. Workaround: Pause campaign, create new campaign with different budget type.
  • Timezone: ‚ùå CANNOT be changed - Set at account level, not per campaign. Note: Display clear warnings about immutable fields during creation and implement pre-edit validation to prevent invalid changes.

Requirement 4: Verify Campaign Creation

User Story: System should check that campaign is created in 3rd party system

What Can Be Done:
  • Sync Verification: ‚úÖ Check Facebook to confirm campaign exists
  • Status Tracking: ‚úÖ Track sync status (pending, synced, failed)
  • Retry Logic: ‚úÖ Handle temporary failures automatically
  • Error Handling: ‚úÖ Provide clear error messages for sync failures
  • Real-time Verification: ‚ö†Ô∏è LIMITATION - Facebook has eventual consistency, making immediate verification unreliable. Changes may take seconds to minutes to propagate, and frequent verification calls may hit rate limits. Recommended approach: Implement asynchronous verification with retry logic and status tracking rather than expecting instant confirmation.
‚úÖ Fully Implementable
